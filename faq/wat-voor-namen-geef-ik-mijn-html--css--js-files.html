Wat voor namen geef ik mijn HTML, CSS, JS files?

De belangrijkste regel is: geef dingen namen die wat betekenen.

Voor HTML is de standaard naam: index.html Als je meerdere HTML files hebt geef andere HTML files dan een betekenisvolle naam: contact.html

Voor CSS is de standaard naam: style.css Als je meerdere CSS files hebt geef andere CSS files dan een betekenisvolle naam: typography.css.

Voor JavaScript is de standaard naam: script.js Als je meerdere JavaScript files hebt geef andere JavaScript files dan een betekenisvolle naam: data_store.js.
