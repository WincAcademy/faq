wat kun je doen met APIs?

Wat je kan doen met een specifieke API ligt aan wat die API aanbiedt. In het algemeen kun je zeggen dat je: * informatie kan ophalen * informatie kan versturen * bestaanden informatie kan updaten * bestaande informatie kan verwijderen

En 'informatie' kan dan van alles zijn, van bioscoopkaartjes, tot bankgegevens, tot Tinder-matches.
