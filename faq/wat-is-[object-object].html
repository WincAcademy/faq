Wat is [Object object] ?

Je kan deze vreemde string tegenkomen als je JavaScript values in HTML stopt. Als zo'n value een string of een nummer is dan gaat het goed:

of
. Maar als je (per ongeluk) een JavaScript object of array daarin stopt dan zet JavaScript dat automatisch voor je om in een string, die string is dan [Object object].
