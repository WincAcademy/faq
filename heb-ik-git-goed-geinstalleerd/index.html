Frequently Asked Questions

Heb ik Git (goed) geïnstalleerd?

Geef op je command line het commando

git —version

Als je daar een boodschap krijgt die je de versie van git vertelt: dan heb je het geïnstalleerd.

Als je een andere boodschap krijgt, dat zal iets zijn van 'ik herken het commando niet', dan heb je Git niet (goed) geïnstalleerd. Kijk dan terug in je opleiding waar je uitlegt krijgt over het installeren van git en volg die stappen.
